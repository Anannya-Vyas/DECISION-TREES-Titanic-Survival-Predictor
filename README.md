## 📝 Description

DECISION-TREES-Titanic-Survival-Predictor is a Python-based machine learning project designed to solve the classic Titanic survival prediction challenge. By leveraging the Decision Tree classification algorithm, the model analyzes passenger demographics and ticket information—such as age, gender, socio-economic status, and cabin location—to determine the likelihood of survival. This project serves as a comprehensive demonstration of the data science lifecycle, covering essential steps from exploratory data analysis and feature engineering to model training and performance evaluation, resulting in a highly interpretable model for binary classification.

## 🛠️ Development Setup

### Python Setup
1. Install Python (v3.8+ recommended)
2. Create a virtual environment: `python -m venv venv`
3. Activate the environment:
   - Windows: `venv\Scripts\activate`
   - Unix/MacOS: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
